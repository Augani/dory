# Dory 0.3.2 — Director's Cut

## Creative idea

**Build the world. Contain the chaos.**

A compact blue Docker Core travels along an amber magnetic buildline, attracting only the parts the user chooses. It opens into a real graphical Linux world, folds into a disposable agent containment chamber, leaves persistent work behind when optional payloads eject, and finally becomes the exact Dory mark. Every effect belongs to the same physical metaphor.

## Narration

Meet Dory zero point three point two.

Your complete local Linux workspace, built for Apple Silicon Macs.

Start with one smaller Docker Core. About a hundred and twelve megabytes of duplicate compatibility payload—gone.

Then add only what you need: Kubernetes, Linux machines, Debian, Ubuntu, or Kali desktops.

Run and test graphical Linux applications beside your containers: Retina-sharp windows, persistent disks, scoped Mac folders, and snapshots.

For A-I agents, launch a dedicated preview sandbox V-M. No host files are visible by default. Mount only what you choose. Set network to none. Roll back. Or let the temporary V-M delete itself.

Optional components are signed and removable. Your workload data stays.

Add custom domains. Add trusted H-T-T-P-S.

Dory. Smaller core. Bigger possibilities. Built for Mac. Open source.

## Sound direction

- Warm, assured narration remains clearly in front.
- Three pitched magnetic contacts become the recurring brand motif.
- Module clicks, portal whoosh, sandbox seal, contained delete crunch, network disconnect, rollback rewind, deletion pop, signed stamp, route pings, and HTTPS lock land in narration gaps.
- Effects never cover feature names or safety claims.
- Master audio is 48 kHz stereo with conservative peaks and enough headroom for social-platform normalization.

## Claim guardrails

- Sandbox remains labelled **Preview**.
- Say and show **no host files by default**, not an absolute promise that an agent can never affect the Mac. Explicit writable mounts remain writable.
- Show only `network none` as isolated. Do not imply the narrower `outbound` request is enforced in 0.3.
- Use a generated sandbox path for the destructive demo, never a host path.
- Claims come from the repository `CHANGELOG.md`, `README.md`, `COMPATIBILITY.md`, and `website/public/agent-guide.json`.
