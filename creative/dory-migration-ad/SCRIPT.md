## Claim provenance and guardrails

- Runtime detection, preserved object types, capacity/collision checks, interrupted-import recovery state, and source non-deletion: root `README.md`, вЂњMove from another runtime.вЂќ
- Discover в†’ Preflight в†’ Import and verify language plus the approved campaign line: `website/src/App.tsx`, migration section.
- Exact migration UI hierarchy: `Dory/Features/Settings/SettingsView.swift`.
- The film says `Dory never deletes the source`; it does not say the source is untouched during every migration phase.
- The visual preflight can show warnings or blockers. It does not imply every workload is immediately portable.
- Fixed ports or live data can require the source workload to stop or pause before a consistent import.
