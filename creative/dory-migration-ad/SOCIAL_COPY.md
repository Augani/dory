## LinkedIn

The hardest part of switching a local Docker runtime is rarely installing the next one. It is trusting that the work already on your machine will come with you.

Dory's Migrate & Compare flow discovers Docker Desktop, OrbStack, Colima, Rancher Desktop, Podman, and other Docker-compatible sockets. It inventories first, checks capacity and name collisions, then performs a recoverable import of the objects the source exposes—including images and tags, named-volume data, custom networks, container configuration, Compose labels, writable layers, ports, and runtime state.

Dory never deletes the source. Keep the old runtime installed until the preflight and post-import checks pass.

Migration is one doorway into Dory: Docker, Compose, Kubernetes, full Linux desktops, persistent servers, recovery, and agent automation in one native Mac app.

Bring the work. Leave the lock behind.
